The AEO (Answer Engine Optimization) Score is a composite metric from 0-100 points that evaluates how well a webpage is optimized for AI-powered answer engines like ChatGPT, Perplexity, and Google's AI Overview.

1. Answerability 30 Points

Measures how well the page directly answers user queries in a consumable, AI-friendly format.

1.1 Direct Answer Presence (12 points)

Detects if the page contains direct, concise answers to likely questions.

Formula: answer_blocks_found = count(answer_patterns) answer_score = min(12, answer_blocks_found √ó 2) Detection Method: - Extract answer patterns: TL;DR, callouts, definition boxes - Count unique answer blocks (excluding blockquotes) - Multiply by 2, cap at 12 points
Answer Blocks Score Rating
0-1 blocks 0-2 points Poor
2-3 blocks 4-6 points Fair
4-5 blocks 8-10 points Good
6+ blocks 12 points Excellent
Example: Page has 4 answer blocks ‚Üí Score = min(12, 4 √ó 2) = 8 points

1.2 Question Coverage (8 points)

Measures how many common user questions the page addresses.

Formula: questions_found = count(heading_questions + inline_questions) coverage_ratio = questions_found / 10 question_score = min(8, coverage_ratio √ó 8) Detection Method: - Extract headings with "?" or question patterns - Count inline questions in content - Normalize to 10 questions baseline

3-53-5 points
6-86-7 points
9+8 points
Example: Page has 7 questions ‚Üí Coverage = 7/10 = 0.7 ‚Üí Score = min(8, 0.7 √ó 8) = 5.6 points

1.3 Answer Conciseness (6 points)

Evaluates if answers are presented in a scannable, AI-friendly format.

Formula: score = 0 // TL;DR section (2 points) if has_tldr_section: score += 2 // Bullet lists (2 points) if list_count >= 3: score += 2 // Short paragraphs (2 points) avg_paragraph_words = sum(paragraph_words) / paragraph_count if avg_paragraph_words <= 100: score += 2 conciseness_score = score (max 6)
Example: Page has TL;DR (2pts) + 4 lists (2pts) + short paragraphs (2pts) = 6 points

1.4 Answer Block Formatting (4 points)

Measures visual formatting that helps AI parse answers.

Formula: score = 0 // Emphasis tags (1 point) if paragraphs_with_emphasis >= 5: score += 1 // Definition/callout boxes (2 points) if has_definition_box OR has_callout_box: score += 2 // Blockquotes (1 point) if has_blockquote: score += 1 formatting_score = score (max 4)

2.4 Completeness of Required Fields (2 points)

Formula: for each schema in schemas: required_fields = get_required_fields(schema['@type']) present_fields = count(required_fields present in schema) completeness = present_fields / total_required_fields avg_completeness = mean(all completeness_scores) completeness_score = avg_completeness √ó 2
